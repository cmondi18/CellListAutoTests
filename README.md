Selenide + Junit4 example
================================

This is a sample project demonstrating how to test Cell List Page (like this http://samples.gwtproject.org/samples/Showcase/Showcase.html#!CwCellList) with Selenide (Selenium webdriver).

**You can checkout and run it locally with a few minutes.**

### How to run with Gradle

Type from command line for running tests:

```
./gradlew test
```
